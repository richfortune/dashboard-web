# Roadmap

## M1 - Frontend Online
- React + Vite setup
- Dashboard layout
- 3 widgets (mock)
- Deploy on Netlify

## M2 - First real data
- Backend .NET API
- One external API (e.g. weather)
- Store data in memory

## M3 - Persistence
- PostgreSQL
- Basic entities
- Save/load data

## M4 - Observability
- Serilog
- Seq
- Health endpoints

## M5 - Async processing
- RabbitMQ
- Worker service
- Background refresh

## M6 - Production
- Docker full stack
- Reverse proxy
- Public deployment
